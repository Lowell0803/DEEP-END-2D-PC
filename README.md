Controls: (1) Left Mouse Button - Attack (2) E - Skill (3) A and D - Move Left and Right (4) Space - Jump (5) Esc - Pause

Note #1: Dying 10 times will prompt a 'Game Over' 

Note #2: Stick to walls then jump to wall jump
